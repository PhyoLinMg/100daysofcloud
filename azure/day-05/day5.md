## Create A Virtual Network IPv4 in Azure

## Steps    

Login to the Portal(as specified in the document)
search for Virtual Network
for the naem, you can use the name specified in the document.

Resource groups , default values are used. 
for the IP address, you can get the ipv4 CIDR from the document.

Click on Review and create.
