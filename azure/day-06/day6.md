## Azure Day 6

## Create A subnet in Azure Virutal Network

## Steps    

Login to the portal of azure as specified in the document.
search for virtual network.

Click on Create Virtual network.

Name - given in document
Resource group - default values are used.

For Ip address range, you can get the ipv4 cidr from the document.

Under subnet, you can click the edit button,  and name the subnet as the document mentioned.
Click Save. 

Click on Review and create.

And DONE.
