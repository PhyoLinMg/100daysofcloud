# Azure - Day 2

## Create an Azure VM

Login to the Azure Console provided by the document.

Select Virtual Machines.

Click on Create Virtual Machine.

Choose Resource Group(There is a default)

Name - given in document

CHoose Image - given in document

VM size - given in document(select `See All Sizes` if you don't see)

Choose SSH or password authentication(not specified in document)

For Storage, choose the image default (30GB)
For Disk Type, Choose as the document specified

After setting up the VM, we should see the VM in the list. We have to try login to the VM using SSH or password.
